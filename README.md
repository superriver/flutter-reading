#### 包含功能：

- 首页轮播图
- 列表上拉加载、下拉刷新
- 底部导航切换保存页面状态
- 页面路由传值
- 换肤功能
- provider状态管理
- 使用sqflite实现收藏功能
- ...
- 持续增加中
